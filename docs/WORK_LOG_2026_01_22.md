## 1. Tổng Quan Task

Điều chỉnh UI/UX, cập nhật quy trình Kanban (3 vòng phỏng vấn), và cấu trúc lại dữ liệu Google Sheet để khớp với yêu cầu thực tế của team TA.

## 2. Các Thay Đổi Chi Tiết

### 2.1. Giao Diện (UI)

- **Header**: Đã ẩn phần hiển thị `User ID` (chỉ còn hiện Email và Role) giúp giao diện gọn gàng hơn.
- **Kanban Board**:
  - Cập nhật tên các cột theo quy trình chuẩn:
    - `New` -> `Test / Screen CV` -> **`HR Interview`** (Mới) -> `Manager L1` -> `Manager L2` -> `Offer`.
  - Logic Modal Phỏng vấn:
    - **Vòng HR Interview**: Chỉ hiện form chọn ngày giờ, KHÔNG bật popup Outlook (vì HR tự quản lý).
    - **Vòng Manager L1/L2**: Bật popup Outlook soạn sẵn email mời như cũ.
  - Logic Withdraw (Rút lui): Cập nhật để khi withdraw từ vòng sau sẽ lùi về đúng vòng trước đó (VD: Withdraw L1 -> Về HR Interview).

### 2.2. Cấu Trúc Dữ Liệu (Backend)

Đã cập nhật Mapping trong code (`route.ts`) để khớp với việc chèn thêm cột "Ngày HR PV". Cấu trúc mới như sau:

| Index | Cột Excel | Trường Dữ Liệu    | Ghi Chú             |

| 31    | **AF**    | `interviewDate1`  | Ngày PV Vòng 1 (L1) |

_Đã tạo file `PLAN_DATA_STRUCTURE.md` lưu lại cấu trúc này._

## 3. Quy Trình Deploy

1.  Code được sửa tại máy local (Visual Studio Code).
2.  Chạy lệnh `git add .` -> `git commit` -> `git push origin main`.
3.  Vercel tự động phát hiện commit mới và tiến hành Build lại trang web.
4.  User truy cập link: `https://cbs-recruitment-web.vercel.app` để thấy thay đổi (sau khoảng 1-2 phút).
